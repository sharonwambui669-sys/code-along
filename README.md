# code-along
This application helps users organize and manage their daily tasks.
The To-Do List application works by allowing users to create and manage tasks using JavaScript objects. The business logic of the project focuses on how tasks are created, stored, updated, and removed from the application.

A constructor function called Task is used to create new task objects. Each task contains:

Task name
Due date
Completion status

When a user enters a task and submits the form, a new task object is created and stored inside an array. The array helps keep track of all tasks added by the user.

Prototype methods are also used in the project. A prototype method called markCompleted() changes the task status from incomplete to completed whenever the user clicks the complete button.

The application also includes functions for:

Displaying tasks on the webpage
Marking tasks as completed
Deleting tasks from the list

The business logic ensures that all task operations work correctly before updating the user interface.
